# 📊 Pizza Sales Dashboard 

An interactive Excel dashboard analyzing pizza sales performance, focusing on revenue trends, order patterns, customer preferences, and product-level insights.

---

## 🎯 Project Overview

This project analyzes pizza sales data using Excel to uncover key business insights such as peak sales periods, best-selling products, and customer ordering behavior. The dashboard enables data-driven decision-making for improving sales and operational efficiency.

---

## 🛠️ Tech Stack

- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Power Query (for data cleaning)  
- Excel Formulas  

---

## 📌 Features / Highlights

### 🔹 Business Problem
Understanding sales performance, customer preferences, and demand patterns to optimize business strategies like pricing, inventory, and promotions.

---

### 🔹 Goal of the Dashboard
To provide a complete overview of pizza sales by analyzing trends, peak hours, product performance, and category contributions.

---

### 🔹 Key Metrics

- Total Revenue  
- Total Orders 
- Average Order Value  
- Total Quantity Sold 
- Avg Pizzas per Order 

---

### 🔹 Key Insights

- 📅 **Busiest Days:** Friday & Saturday  
- ⏰ **Peak Hours:** 12 PM – 1 PM and 5 PM – 8 PM  
- 🍕 **Top Category:** Classic pizzas contribute the highest sales  
- 📏 **Top Size:** Large size pizzas dominate revenue share  
- 🏆 **Best Seller:** The Classic Deluxe Pizza  
- ⚠️ **Worst Seller:** The Brie Carre Pizza  
- 📈 Sales show consistent demand with clear peak time patterns  

---

### 🔹 Dashboard Features

- Daily sales trend analysis  
- Hourly order distribution  
- Category-wise sales contribution  
- Size-wise performance breakdown  
- Top 5 and Bottom 5 pizzas by sales  
- Interactive date filter (timeline slicer)  

---

## ⚠️ Challenges Faced

- Cleaning and structuring raw Excel data  
- Creating dynamic dashboards using Pivot Tables  
- Handling time-based analysis (hourly trends)  
- Designing a visually appealing layout in Excel  

---

## 📎 How to Use

1. Download the Excel file (`.xlsx`)  
2. Open in Microsoft Excel  
3. Use slicers and filters to explore the dashboard  
4. Analyze trends and insights interactively
